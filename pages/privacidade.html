Política de Privacidade

Conformidade com LGPD. Seus dados estão seguros.

1. Introdução

A QG Digital ("nós", "nosso" ou "LocalInsta") está comprometida com a proteção de seus dados pessoais. Esta Política de Privacidade explica como coletamos, usamos, divulgamos e protegemos suas informações de acordo com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

2. Dados que Coletamos

O LocalInsta coleta os seguintes dados:

  • Credenciais do Instagram (armazenadas criptografadas no seu PC)
  • Fotos e conteúdo que você adiciona à galeria
  • Horários de publicação configurados
  • Email fornecido no formulário de contato
  • Informações de uso do app (para melhorias)

3. Onde os Dados São Armazenados

Local-First: Todos os seus dados sensíveis (credenciais, fotos, configurações) são armazenados apenas no seu computador. Nós não mantemos cópias em servidores externos. Suas informações nunca saem do seu PC sem sua permissão explícita.

4. Como Usamos Seus Dados

Seus dados são usados para:

  • Executar a automação do Instagram conforme configurado
  • Melhorar a qualidade do LocalInsta
  • Responder a suas mensagens de contato
  • Cumprir obrigações legais

5. Compartilhamento de Dados

Nós não compartilhamos seus dados pessoais com terceiros, exceto:

  • Quando exigido por lei ou ordem judicial
  • Para proteger nossos direitos legais
  • Com seu consentimento explícito

6. Segurança dos Dados

Implementamos medidas de segurança para proteger seus dados:

  • Criptografia: Credenciais são criptografadas no seu PC
  • Sem Nuvem: Dados não são sincronizados com servidores
  • Controle Total: Você tem controle total sobre seus dados
  • Atualizações: Recebemos atualizações de segurança regularmente

7. Seus Direitos (LGPD)

Você tem direito a:

  • Acesso: Solicitar acesso aos seus dados
  • Correção: Corrigir dados imprecisos
  • Exclusão: Solicitar exclusão de dados (exceto aqueles necessários por lei)
  • Portabilidade: Receber seus dados em formato estruturado
  • Oposição: Opor-se ao processamento de dados

8. Cookies e Rastreamento

O LocalInsta não usa cookies ou tecnologias de rastreamento. O site pode usar análises básicas para entender como você o usa, mas sem identificação pessoal.

9. Retenção de Dados

Mantemos seus dados apenas enquanto você usar o LocalInsta. Quando você desinstala o app, todos os dados locais são removidos. Dados de contato (email) são mantidos por 1 ano para fins de suporte.

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos você sobre mudanças significativas via email ou no site.

11. Contato

Se você tiver dúvidas sobre esta Política de Privacidade ou sobre como seus dados são tratados, entre em contato:

Email: atendimento.qgdigital@gmail.com

12. Conformidade com LGPD

O LocalInsta está em conformidade com a Lei Geral de Proteção de Dados (LGPD). Você é o controlador de seus dados, e nós somos o processador. Todos os princípios da LGPD são respeitados:

  • ✓ Finalidade: Dados usados apenas para automação do Instagram
  • ✓ Adequação: Coletamos apenas dados necessários
  • ✓ Necessidade: Minimizamos coleta de dados
  • ✓ Livre acesso: Você tem acesso total aos seus dados
  • ✓ Qualidade: Mantemos dados precisos e atualizados
  • ✓ Transparência: Esta política explica tudo claramente
  • ✓ Segurança: Implementamos medidas de proteção
  • ✓ Responsabilidade: Somos responsáveis pelo tratamento de dados
